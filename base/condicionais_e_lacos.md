# Condicionais e Laços em Python

De forma geral, podemos dizer que é possível alterar o comportamento do código de acordo com condições previamente determinadas, **como quando certas condições são atendidas ou repetidas enquanto uma condição for verdadeira**. Isso é possível graças ao uso das **condicionais e laços de repetição**.

## 1. Condicionais (`if`, `elif`, `else`)
As condicionais são usadas para tomar decisões dentro do código. O programa avalia expressões lógicas e executa diferentes blocos de código de acordo com o resultado:

- `if` → executa um bloco se a condição for verdadeira
- `elif` → avalia uma nova condição, caso a anterior seja falsa
- `else` → executa quando nenhuma das condições anteriores é satisfeita

## 2. Operadores Lógicos
As condicionais usam operadores lógicos para combinar certas condições:

- `and` → todas as condições devem ser verdadeiras
- `or` → pelo menos uma condição deve ser verdadeira
- `not` → inverte o valor lógico de uma condição

## 3. Laços de repetição
Os laços nos permitem repetir uma ação automaticamente, evitando repetição manual de código:

- `while` → O laço while executa um bloco de código enquanto uma condição for verdadeira.
É muito usado para validação de entrada de dados e execução controlada por condições.

- `for` → O laço for é utilizado para percorrer sequências, como listas, strings ou intervalos numéricos, executando um bloco de código a cada iteração.

Tudo isso pode ser visto na prática nos códigos `condicionais_e_lacos1.py` e `condicionais_e_lacos2.py`.
